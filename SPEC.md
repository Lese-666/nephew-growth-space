# 简历生成器 - 规格说明文档

## 1. 项目概述

**项目名称**: 简历生成器 (Resume Generator)
**项目类型**: 单页面 Web 应用
**核心功能**: 用户在左侧输入简历信息，右侧实时预览，支持一键导出 PDF/图片
**目标用户**: 求职者、学生、职场人士

---

## 2. 技术方案

### 核心技术栈
- **HTML5 + CSS3 + Vanilla JavaScript** (纯前端实现，无需后端)
- **html2canvas**: 将简历 DOM 转换为图片
- **jsPDF**: 生成 PDF 文件
- **Google Fonts**: 字体支持

```

├── SPEC.md            # 规格说明文档
```

## 3. 功能列表

### 3.1 简历信息输入 (左侧面板)
- **基本信息** (可选)
  - 姓名
  - 电话
  - 邮箱
  - 证件照上传 (支持拖拽)
  - 求职意向
  - 地址

- **学历信息** (可选，可添加多条)
  - 学校名称
  - 学历 (高中/本科/硕士/博士)
  - 专业
  - 在校时间

- **项目实践** (可选，可添加多条)
  - 项目名称
  - 项目时间
  - 项目描述

- **擅长技能** (可选)
  - 技能列表 (逗号分隔)

- **自我评价** (可选)
  - 评价文本

### 3.2 简历预览 (右侧面板)
- 实时响应左侧输入变化
- 空板块不显示
- A4 纸张比例展示

### 3.3 主题与自定义
- **内置主题** (3种)
  - 经典专业 (蓝色系)
  - 简约清新 (绿色系)
  - 商务精英 (深灰系)

- **自定义选项**
  - 主题颜色选择
  - 字体选择 (默认/微软雅黑/思源宋体/等线)
  - 字号大小调整

### 3.4 外观模式
- 浅色模式 (默认)
- 暗色模式 (支持切换)

### 3.5 导出功能
- **导出为 PDF**
- **导出为图片 (PNG)**
- 导出的文件保存到本地

## 7. 交互规范

### 7.1 实时预览
- 输入框 `input` 事件触发预览更新
- 延迟 100ms 防抖

### 7.2 文件上传
- 支持点击上传或拖拽上传
- 支持格式: JPG, PNG, WEBP
- 图片预览: 转为 Base64 显示
- 最大尺寸: 2MB

### 7.3 导出流程
1. 点击导出按钮
2. 使用 html2canvas 将预览区域转为 canvas
3. 如果是 PDF: 用 jsPDF 包装 canvas
4. 如果是图片: 直接下载 PNG
5. 触发浏览器下载

### 7.4 主题切换
- 点击切换按钮平滑过渡 (CSS transition 0.3s)
- 记住用户偏好 (localStorage)
